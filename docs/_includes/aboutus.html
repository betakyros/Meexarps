One thing I learned while working on this project is that things take time and you can't publish a game on passion alone. You'll need a good chunck of discipline to get through the hard times.

Hey, I'm Regina, and I helped with the UX and initial character illustrations for this game. It's been a rewarding journey working on this game and seeing the wonderful reactions from playtesters, and I learned a lot about how to take incremental feedback to continuously improve UX. It's been a long process, but now that this is over, you can catch me on adventures outside.
